# 📜 Master Document: Ant Farm Observatory
**System Status:** 🟢 GOLD (Reference Implementation)
**Version:** v1.0

## 1. Concept and General Architecture
Ant Farm Observatory is a two-layer system:
- **Engine Layer**: Dwarf Fortress Classic v50.14 running 24/7 (Webtop).
- **Observatory Layer**: Custom web app that reads the legends.xml file.

## 2. Port Map
- **8080**: Webtop (DF desktop).
- **8090**: Chronicles of the World (Observatory).
