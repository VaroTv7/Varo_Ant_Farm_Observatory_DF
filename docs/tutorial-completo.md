# 📜 Documento Maestro: Ant Farm Observatory
**Estado del Sistema:** 🟢 ORO (Implementación de Referencia)
**Versión:** v1.0

## 1. Concepto y arquitectura general
Ant Farm Observatory es un sistema de dos capas:
- **Capa motor**: Dwarf Fortress Classic v50.14 corriendo 24/7 (Webtop).
- **Capa observatorio**: Aplicación web propia que lee el archivo legends.xml.

## 2. Mapa de Puertos
- **8080**: Webtop (escritorio DF).
- **8090**: Crónicas del Mundo (Observatorio).

## 3. Instalación rápida
```bash
./install.sh
```
